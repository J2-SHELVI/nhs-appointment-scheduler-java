# nhs-appointment-scheduler-java
Java Swing desktop application for NHS-style appointment scheduling with CRUD operations, search, and rule-based priority system.

## Features
- Login system (admin/admin123)
- Add, view, update, delete appointments
- Search functionality
- Rule-based priority system (Emergency = High, Missed = Medium)

## Tech Stack
- Java
- Java Swing
- IntelliJ IDEA

## How to Run
1. Open the project in IntelliJ IDEA
2. Run `Main.java`
3. Login using:
   - Username: admin
   - Password: admin123
